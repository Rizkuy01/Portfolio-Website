## Introduction

This portfolio website is designed and developed by me. It serves as a platform to showcase my skills as a fullstack web developer and display my recent projects and resume.

## Features

1. **Responsive Design**: The website is fully responsive and optimized for various devices and screen sizes.
2. **Resume Page**: A dedicated page where visitors can view and download my resume.
3. **Projects Page**: A page showcasing my recent projects along with details and links to GitHub repositories or live demos.
4. **Bootstrap Integration**: Utilizes Bootstrap for streamlined and responsive CSS styling.
5. **Interactive Elements**: Incorporates JavaScript for interactive elements and enhanced user experience.

## Technologies Used

- HTML
- CSS (Bootstrap)
- JavaScript

## Setup

To run the website locally, follow these steps:

1. Clone this repository to your local machine.
2. Navigate to the project directory.
3. Open `index.html` in your preferred web browser.

## Usage

Feel free to explore the website and its features. Here's a brief overview:

- **Home**: Introduction to My branding and links to other pages.
- **Resume**: View and download my resume.
- **Projects**: Browse through recent projects with links to GitHub repositories or live demos.
